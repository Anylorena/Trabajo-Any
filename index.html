¿Qué es ECMAScript?

ECMAScript es una especificación creada por Ecma International que define cómo debe funcionar el lenguaje JavaScript. Es la base estándar que siguen los navegadores para asegurar compatibilidad entre ellos.

Versiones de ECMAScript

  • ES1 (1997): Primera versión.
  • ES3 (1999): Añade expresiones regulares.
  • ES5 (2009): Strict mode, JSON.parse/stringify, funciones array como forEach, map.
  • ES6 / ES2015: let, const, arrow functions, clases, template literals.
  • ES7 / ES2016: Includes, exponentiation operator.
  • ES8 / ES2017: Async/await, Object.entries.
  • ES9 / ES2018: Rest/spread en objetos, Promise.finally.
  • ES10 / ES2019: flat(), trimStart(), trimEnd().
  • ES11 / ES2020: Nullish Coalescing, Optional Chaining.
  • ES12 / ES2021: replaceAll, Logical Assignment Operators.
  • ES13 / ES2022: Top-level await, at().
  • ES14 / ES2023: Array findLast(), Hashbang grammar.

¿Cómo usar ECMAScript?

Solo necesitas un navegador moderno o un entorno como Node.js. Asegúrate de que tu entorno soporte la versión que deseas usar.

Diferencias entre versiones y ejemplos

ES6: Arrow Functions

const suma = (a, b) => a + b;

ES7: includes()

const nums = [1, 2, 3]; nums.includes(2); // true

ES8: async/await


async function obtenerDatos() {
  const res = await fetch("https://api.example.com");
  const data = await res.json();
  console.log(data);
}

ES9: spread en objetos


const obj1 = { a: 1 };
const obj2 = { ...obj1, b: 2 };

ES10: Array.flat()


const arr = [1, [2, [3]]];
console.log(arr.flat(2)); // [1, 2, 3]
